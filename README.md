# Customer Churn Prediction & LTV Engine

## Project Overview
This project predicts customer churn for a telecom company using Python, PostgreSQL, and Machine Learning.

## Technologies Used
- Python
- PostgreSQL
- Pandas
- Scikit-Learn
- SQLAlchemy
- Matplotlib
- Seaborn

## Dataset
Telco Customer Churn Dataset from Kaggle.

## Features
- Data cleaning
- Exploratory Data Analysis (EDA)
- Churn prediction
- Customer Lifetime Value analysis

## Project Structure
- eda.py → Data analysis
- WA_Fn-UseC_-Telco-Customer-Churn.csv → Dataset
- table_creation.sql → SQL table creation
- importing_dataset.sql → Data import queries
